## 🛠 详细执行步骤

### 1. 文本解析与预处理 (`book-parser`)
*   **章节识别**：利用增强正则识别 `Chapter n`、`第n章`、`【第n章】` 等多种格式。
*   **碎片化存储**：将解析出的章节保存至 `output/chapters/`，方便后续 AI 分片读取，避免上下文溢出。
*   **任务创建**：生成 `.task.json` 记录所有 API 配置和参数。

### 2. AI 核心 DNA 提取
*   **核心内容**：核心 Hook、情节划分、情绪曲线、人物冲突。
*   **产出**：`book-analysis.json`。

### 3. 生成自主受众画像 (Enriched Persona)
*   **维度**：人口、心理触发、阅读行为、审美画像、社会化路径。
*   **产出**：`audience-profile-reverse-engineered.json`。

### 4. 产出适配卖点 (DNA + 人群)
*   **逻辑**：根据“书有什么”+“谁在看”，提炼高转化差异化卖点。
*   **产出**：`selling-points-compare.json`。

### 5. 产出创意方向 (DNA + 人群 + 卖点)
*   **创意爆发**：生成广告脚本、视觉 Brief、核心钩子（Hook）。
*   **产出**：`ad-creative-directions.json`。

### 6. 报告自动合成 (`report-generator`)
*   **数据聚合**：整合所有分散在 output 目录的 JSON 模块。
*   **可视化**：将评分映射为星级 (⭐)，生成排版精美的 Markdown 报告。
*   **产物**：[MARKETING-REPORT.md](output/MARKETING-REPORT.md)。

## 💡 如何执行

**启动全链路自动化：**
```bash
./book-parser analyze "path/to/book.txt" --auto --compare --reverse-engineer
```

**快速查看结果：**
```bash
cat output/MARKETING-REPORT.md
```
