# Book Parser Skill

AI 驱动的书籍智能解析系统，为国际市场书籍营销提供精准洞察。

## 功能

1. **章节解析**：自动识别章节，支持多种格式
2. **AI 分析**：一键生成完整的营销分析报告
3. **相似书籍查询**：Google Books API
4. **广告数据查询**：AnyNovel API

## 使用方法

### 通过 OpenClaw 调用（推荐）

```
"帮我分析这本书：<book.txt>"
```

AI 会自动：
1. 解析章节
2. 生成分析提示词
3. 调用 Claude 生成完整分析
4. 保存结果到输出目录
