## Step 2: DNA 提取 - 书籍核心基因

### 原版维度（8个主要维度）
```
basicInfo (5个字段)
bookDNA (4个曲线)
addictionIndex (5个因素 + 预测)
characters (主角 + 配角 + 关系)
plot (结构 + 冲突 + 章节分解)
goldenQuotes
highlights
preliminarySellingPoints
```

### 增强版维度（11个主要维度）
```
✅ basicInfo (扩展到 9个字段)
   + themes (主题)
   + toneStyle (基调)
   + targetMarkets (目标市场)
   + competitiveBooks (竞品)

✅ bookDNA (扩展到 6个曲线)
   + tensionCurve (张力曲线)
   + satisfactionPoints (满足点)
   每个数据点增加更多上下文

✅ addictionIndex (扩展)
   + 每个因素增加详细分析
   + viralPotential (病毒传播潜力)
   + wordOfMouthScore (口碑评分)
   + strengths (优势列表)

✅ characters (大幅扩展)
   主角增加:
   + strengths/weaknesses (优缺点)
   + desires/fears (欲望/恐惧)
   + backstory (背景故事)
   + visualDescription (视觉描述)
   
   配角增加:
   + importance (重要性)
   + likability (好感度)
   + function (功能)
   + visualDescription
   
   新增:
   + characterDynamics (角色动态)

✅ plot (扩展)
   + subplots (支线)
   + resolution (结局类型)
   + turningPoints (转折点)
   + foreshadowing (伏笔)
   章节分解增加:
   + keyEvents (关键事件)
   + emotionalBeat (情感节拍)
   + cliffhanger (悬念)
   + coolPoints (爽点)

🆕 writingStyle (新增维度)
   + language (语言风格)
   + dialogueQuality (对话质量)
   + descriptionDensity (描述密度)
   + pacing (节奏感)
   + readability (可读性)
   + uniqueFeatures (独特特色)

🆕 marketingAngles (新增维度)
   + goldenQuotes (扩展，增加上下文和病毒潜力)
   + highlights (扩展，增加视觉和音效建议)
   + preliminarySellingPoints (扩展，增加平台和CTR预估)
   + viralElements (病毒元素)
   + memePotential (梗潜力)

🆕 contentWarnings (新增维度)
   + sensitiveTopics (敏感话题)
   + ageRating (年龄分级)
   + culturalConsiderations (文化考量)
```

**数据点增加**: 约 **150% ↑**

## Step 3: 受众画像生成

```

```

```
🆕 competitiveInsights (新增维度)

🆕 persona (新增)
```

## Step 4: 卖点提炼

### 原版维度
```
finalSellingPoints (每个卖点 5个字段)
  - title
  - oneLinePitch
  - priority
  - psychologicalTriggers

```

### 增强版维度
```
✅ finalSellingPoints (扩展到 13个字段)
   + extendedPitch (扩展文案)
   + emotionalAppeal (情感诉求)
   + targetSegment (目标细分)
   + uniqueAngle (独特角度)
   + proofPoints (支撑点)
   
   scoring 扩展到 8个评分:
   + credibility (可信度)
   + memorability (记忆度)
   + shareability (分享度)
   + conversionPotential (转化潜力)
   
   + platformFit (平台适配度)
   + testingRecommendations (测试建议)
   + risks (风险)
   + opportunities (机会点)

✅ competitiveAdvantage (扩展到 4个字段)
   + differentiators (差异化点)
   + marketGap (市场空白)
   + positioningStatement (定位陈述)

🆕 messagingHierarchy (新增维度)

   + supportingMessages (支撑信息)

🆕 contentPillars (新增维度)
```
