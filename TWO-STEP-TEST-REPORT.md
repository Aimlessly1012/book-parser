## 💡 技术创新

### 1. 两步提取策略
```javascript
// Step 2a: 提取每章梗概（每章只取前2000字）
章节1-8 → 梗概提取 → 8个结构化梗概

// Step 2b: 基于梗概分析
8个梗概 → DNA 分析 → 完整书籍 DNA
```

**优势**:
- 输入小（每章2000字 vs 完整章节）
- 覆盖全（8章 vs 3章）
- 速度快（结构化数据 vs 长文本）

### 2. 数据复用
```
章节梗概 ──┬→ DNA 分析
           ├→ 卖点提炼（直接使用）
           └→ 创意方向（间接使用）
```

避免重复传递大量数据，减少 token 消耗。

### 3. 降级方案
```javascript
if (章节梗概存在) {
  使用梗概（最优）
} else {
  使用 DNA（降级）
}
```

保证兼容性，支持多种模式。
